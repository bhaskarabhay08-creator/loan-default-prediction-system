# AI-Powered Loan Default Prediction System

## Overview
An AI-powered web application developed during my Summer Internship at NIELIT Ranchi.

This system uses Machine Learning techniques to analyze applicant data and predict the likelihood of loan default, helping financial institutions make informed lending decisions.

## Features
- Loan Default Risk Prediction
- User Authentication
- Interactive Dashboard
- Data Visualization
- AI/ML-Based Decision Support
- Modern Responsive UI

## Technologies Used
- Python
- Machine Learning
- Next.js
- TypeScript
- Tailwind CSS
- PostgreSQL

## Project Objective
To assist banks and financial institutions in identifying high-risk loan applicants using predictive analytics and machine learning.

## Internship
Developed during the Summer Internship Program in Artificial Intelligence using Python at NIELIT Ranchi.
